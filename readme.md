## eprice
a command line tool for ethereum price exchanges

### Installation
~~~bash
  npm i eprice -g
~~~

### Shortcuts

~~~
[enter]  =>  refresh prices 
[o]      =>  open site in browser
[q]      =>  quit
~~~

### Supported Exchanges
- [bitfinex](https://bitfinex.com/)
- [etherscan](https://etherscan.io)
- [poloniex](https://poloniex.com)
- [kraken](https://kraken.com)
